# kraken-api-test
Verify basic Kraken API features

# Prerequisites

- NodeJS 12+ installed
- npm 6+ installed

# How to run

- `npm install`
- `npm run test` or `npx cypress run`

# Test results

- Using built-in Mocha Spec test reporter
- Screenshots on fail will be saved to `screenshots` folder
